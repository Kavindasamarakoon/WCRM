# VersaWA - WhatsApp CRM & Marketing Platform

Enterprise WhatsApp Business Platform management system.

## Tech Stack

- Next.js 15 + TypeScript
- NestJS API
- PostgreSQL + Prisma
- Redis + BullMQ
- Docker
